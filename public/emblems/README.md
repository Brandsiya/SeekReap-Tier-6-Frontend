# SeekReap Emblems & Logos

This directory contains all SeekReap branding assets for the frontend.

## Directory Structure

- **primary/** - Main logos for headers and footers
- **favicons/** - Browser favicons and app icons
- **social/** - Social media sharing images
- **certificates/** - Logos for PDF certificates
- **qr-overlay/** - Logo overlays for QR codes
